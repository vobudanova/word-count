# 100 слов

Плагин WordPress, который показывает рядом с записью небольшой блок: **сколько времени займёт чтение** и **сколько в тексте слов**.

Например: **«3 мин.»** и **«520 слов»**.

Время считается по средней скорости чтения (200 слов в минуту). Подсчёт слов работает на любом языке — кириллица, латиница, иероглифы — и правильно склоняет «слово / слова / слов». Интерфейс и вывод доступны на **русском и английском** и переключаются автоматически вслед за языком сайта.

---

## Что умеет плагин

- Показывать блок в разных местах сайта — вы сами выбираете, где именно.
- В каждом месте показывать что нужно: **время чтения**, **количество слов** или **и то, и другое** (или ничего).
- **Отдельно настраивать посты и страницы** — это в WordPress разные вещи, и плагин их различает.
- Переопределять вывод **для конкретной записи** прямо в редакторе — например, спрятать блок на одной странице.
- Вставлять блок вручную в любом месте через шорткод `[reading_time]`.

## Установка

1. Скачайте плагин ZIP-архивом (кнопка **Code → Download ZIP** на GitHub).
2. В админке WordPress откройте **Плагины → Добавить новый → Загрузить плагин**, выберите архив и нажмите **Установить**.
3. Нажмите **Активировать**.
4. В левом меню появится пункт **Время чтения** (иконка часов) — там все настройки.

> Или распакуйте папку плагина в `/wp-content/plugins/` по FTP и активируйте на странице «Плагины».

---

## Как настроить

Откройте **Время чтения** в левом меню админки. Вы увидите пять строк — это **места**, где может появляться блок. Для каждого места выбирается один из четырёх вариантов:

- **Не показывать**
- **Только время**
- **Только количество слов**
- **Время + количество слов**

### Пять мест

| Место | Где именно показывается |
|---|---|
| **После заголовка поста** | На странице поста, сразу под заголовком |
| **В теле поста** | На странице поста, перед основным текстом |
| **В теле страницы** | На обычной странице (не посте), перед текстом |
| **В списках и архивах** | На главной, в рубриках и результатах поиска — у анонса каждого поста |
| **В связанных постах** | Рядом с заголовками записей в блоках вроде «Похожие записи» |

Нажмите **Сохранить изменения** — и всё применится сразу.

> **Совет.** «После заголовка поста» и «В теле поста» — это два *разных* места на одной и той же странице. Если включить оба, блок появится дважды. Обычно выбирают что-то одно.

## Настройка для отдельной записи

Иногда нужно, чтобы на одной конкретной записи блок вёл себя не как везде. Это можно сделать, не трогая общие настройки.

Откройте пост или страницу на редактирование — в боковой панели редактора найдите блок **Время чтения** с выпадающим списком:

- **По умолчанию** — использовать общие настройки (обычное поведение).
- **Не показывать** — спрятать блок на этой записи.
- **Только время** / **Только количество слов** / **Время + количество слов** — показать именно это.

Эта настройка действует **только на саму эту запись** (её страницу). На то, как запись выглядит в списках и в «похожих», она не влияет — там работают общие настройки.

---

## Ручная вставка: шорткод

Если хотите поставить блок в произвольное место записи, вставьте шорткод:

```
[reading_time]
```

По умолчанию показывает время и количество слов. Можно ограничить:

```
[reading_time show="time"]     — только время чтения
[reading_time show="words"]    — только количество слов
[reading_time show="both"]     — и то, и другое (по умолчанию)
```

## Языки

Плагин переведён на **русский** и **английский**. Ничего настраивать не нужно — язык интерфейса и вывода совпадает с языком сайта (**Настройки → Общие → Язык сайта**).

## Частые вопросы

**Можно ли изменить скорость чтения?**
Пока нет. Используется средняя скорость — 200 слов в минуту.

**Почему на странице блок не появляется, хотя для постов работает?**
Пост и страница настраиваются отдельно. Проверьте строку **«В теле страницы»** в настройках — по умолчанию она выключена.

**Как убрать блок только с одной записи?**
Откройте её в редакторе и в панели **Время чтения** выберите **Не показывать**.

**Работает ли с произвольными типами записей?**
Автоматический вывод рассчитан на посты и страницы. Для остального используйте шорткод `[reading_time]` — он работает где угодно.

**Что будет с моими настройками при деактивации?**
Ничего — они сохраняются. Настройки и данные удаляются только при полном удалении плагина.
