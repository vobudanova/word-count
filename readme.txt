Отвечаю за слова.

== Description ==

Отвечаю за слова с детства.

Плагин показывает рядом с постом блок «N мин. / N слов»: ориентировочное время чтения (200 слов в минуту) и количество слов. Поддерживает кириллицу, склоняет «слово / слова / слов» по правилам русского языка и разделяет тысячи узким неразрывным пробелом.

**Возможности:**

* Автоматический вывод после заголовка или перед контентом поста.
* Шорткод `[reading_time]` для ручной вставки в любом месте.
* Опция «Показывать в связанных постах» — выводить блок рядом со ссылками в виджете «Похожие записи».
* Опция «Показывать количество слов» — выводить только время чтения или вместе с количеством слов.
* Корректный подсчёт слов на любом языке (кириллица, латиница, CJK).

== Installation ==

1. Загрузите папку плагина в `/wp-content/plugins/`.
2. Активируйте плагин на странице «Плагины» в админке WordPress.
3. Откройте пункт меню **Время чтения** (иконка часов в боковом меню).
4. На странице **Настройки времени чтения и количества слов** выберите нужные опции.
5. По желанию вставьте шорткод `[reading_time]` в любое место поста или шаблона темы.

== Frequently Asked Questions ==

= Можно ли изменить скорость чтения? =

В этой версии — нет. По умолчанию используется средняя скорость 200 слов в минуту.

= Работает ли с произвольными типами записей? =

Шорткод `[reading_time]` — да. Автоматический вывод после заголовка работает только для типа `post`.
