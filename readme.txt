Показывает рядом с записью время чтения и количество слов. Гибко настраивается по местам вывода, поддерживает русский и английский.

== Description ==

Плагин показывает рядом с записью небольшой блок: сколько времени займёт чтение и сколько в тексте слов. Например: «3 мин.» и «520 слов».

Время считается по средней скорости чтения (200 слов в минуту). Подсчёт слов работает на любом языке — кириллица, латиница, иероглифы — и правильно склоняет «слово / слова / слов». Интерфейс и вывод доступны на русском и английском и переключаются автоматически вслед за языком сайта.

**Возможности:**

* Показ блока в разных местах сайта — вы сами выбираете, где именно.
* В каждом месте показывать нужное: время чтения, количество слов или и то, и другое (либо ничего).
* Отдельная настройка для постов и страниц — плагин их различает.
* Переопределение вывода для конкретной записи прямо в редакторе.
* Ручная вставка в любом месте через шорткод `[reading_time]`.

**Где может выводиться блок:**

* После заголовка поста — на странице поста, сразу под заголовком.
* В теле поста — на странице поста, перед основным текстом.
* В теле страницы — на обычной странице, перед текстом.
* В списках и архивах — на главной, в рубриках и поиске, у анонса каждого поста.
* В связанных постах — рядом с заголовками записей в блоках вроде «Похожие записи».

Для каждого места выбирается один из вариантов: не показывать, только время, только количество слов, время + количество слов.

== Installation ==

1. Скачайте плагин ZIP-архивом и в админке откройте «Плагины → Добавить новый → Загрузить плагин», либо загрузите папку плагина в `/wp-content/plugins/` по FTP.
2. Активируйте плагин на странице «Плагины».
3. Откройте пункт меню **Время чтения** (иконка часов в боковом меню).
4. На странице настроек выберите, что показывать в каждом месте, и сохраните изменения.
5. По желанию вставьте шорткод `[reading_time]` в любое место записи или шаблона темы.

== Frequently Asked Questions ==

= Можно ли изменить скорость чтения? =

Пока нет. Используется средняя скорость — 200 слов в минуту.

= Почему на странице блок не появляется, хотя для постов работает? =

Пост и страница настраиваются отдельно. Проверьте пункт «В теле страницы» в настройках — по умолчанию он выключен.

= Как убрать блок только с одной записи? =

Откройте её в редакторе и в панели **Время чтения** выберите «Не показывать».

= Можно ли показать только время или только количество слов в шорткоде? =

Да: `[reading_time show="time"]` — только время, `[reading_time show="words"]` — только количество слов, `[reading_time show="both"]` — и то, и другое (по умолчанию).

= Работает ли с произвольными типами записей? =

Автоматический вывод рассчитан на посты и страницы. Для остального используйте шорткод `[reading_time]` — он работает где угодно.

= Что будет с настройками при деактивации? =

Они сохраняются. Настройки и данные удаляются только при полном удалении плагина.
